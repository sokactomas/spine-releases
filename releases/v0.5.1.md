# Spine v0.5.1

Released: 2026-06-22

## What's new

- Advisor health is visible: Stats now records real advisor calls separately from settings test pings, so you can see whether Spine's second-opinion path is actually being used in practice and whether it is succeeding.
- Test advisor from Settings: Advisor settings now has a one-click live test that runs through the currently selected provider and model, then shows the result immediately instead of making you verify it manually in a terminal.
- Provider readiness cards: Advisor settings now shows lightweight readiness for Codex, Claude, and local models, so install, login, or local-server gaps are visible before you try a real escalation.
- Privacy shows last advisor status: Privacy now surfaces the latest advisor result and split counts for real calls versus test runs, which makes the setup state legible without opening Stats.
- Narrower advisor examples: Advisor settings now includes copyable example prompts for architecture tradeoffs, migration risk review, and debugging second opinions to encourage focused escalations instead of broad session dumps.

## Install

1. Download the `Spine_0.5.1_universal.dmg` asset from GitHub Releases.
2. Open the DMG and drag Spine to Applications.
3. Launch Spine and restart your CLI if it was already running.

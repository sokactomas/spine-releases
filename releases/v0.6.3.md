# Spine v0.6.3

Released: 2026-07-13

## What changed

This release makes the shared MCP bridge more reliable without making Spine
heavier.

- **Delegation no longer holds one MCP request open.** `spine_delegate` starts
  a bounded job and returns immediately; `spine_delegate_status` reports its
  progress, final output and requested-model verification. Jobs are cancelled
  if their MCP client disconnects, so waiting work cannot become orphaned.
- **The graph accepts cleaner knowledge.** Facts, decisions and corrections now
  require useful content. Tags are normalized and de-duplicated before they are
  written to the shared graph.
- **Activity is named honestly.** The menu-bar badge counts active agent
  sessions, including delegated workers, rather than implying each count is a
  separate terminal window.
- **Notes and diagnostics are sturdier.** Older notes with empty tag data no
  longer blank the Notes screen; local-work metrics and raw diagnostics use
  clearer, non-billing language.
- **About is lighter.** Spine now describes itself plainly as a lightweight
  shared memory and MCP bridge - your terminal remains the workspace.

## Install

1. Download `Spine_0.6.3_universal.dmg` from this release.
2. Open the DMG and drag Spine to Applications.
3. Launch Spine from Applications.
4. Restart any running Claude, Codex or Antigravity sessions so they load the
   updated MCP tool list.

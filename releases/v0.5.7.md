# Spine v0.5.7

Released: 2026-07-02

## What's new

This release keeps the local model lite by default and tidies up the Review flow.

- Local model stays lite: the local model now clearly defaults to embeddings-only — the tiny embedding path is marked recommended, and local chat generation is flagged as the heavy, opt-in path with a CPU warning. Leave the chat model empty and generation stays on the capped cloud CLI, so enabling a local model can't accidentally overwhelm a weaker machine.
- Review, tidied up: the Review count badge no longer gets clipped at high counts, the Review Changes dialog gained a Select all / Deselect all toggle so you can merge everything in one move, and the Settings section headings line up cleanly with their rows.

## Install

1. Download the `Spine_0.5.7_universal.dmg` asset from GitHub Releases.
2. Open the DMG and drag Spine to Applications.
3. Launch Spine and restart your CLI if it was already running.

# Spine v0.5.0

Released: 2026-06-22

## What's new

- Calmer default look: Spine now defaults to the quieter Graphite Lab palette and the refreshed theme set is more even overall, so long sessions feel less harsh than the older high-contrast dark variants.
- Richer CLI stats: Stats now breaks usage down by CLI and model with tokens by CLI, most-used models, and a per-CLI model summary for Claude and Codex.
- Codex projection health: the Codex projection panel now tells you when counts are current versus still reflecting a pre-cap snapshot.
- Better Codex model detection: Spine now extracts Codex model names from transcript context more reliably, so `/stats` can attribute Codex activity to real models.
- Split panes keep cwd: when you split a Spine terminal, the new pane starts in the same working directory as the pane you split.

## Install

1. Download the `Spine_0.5.0_universal.dmg` asset from GitHub Releases.
2. Open the DMG and drag Spine to Applications.
3. Launch Spine and restart your CLI if it was already running.

# Spine v0.5.5

Released: 2026-07-02

## What's new

This release makes Spine feel lite again — a simpler Settings, a readable Skills tab, and no more white-screen crashes.

- Settings went lite: the Advanced fold is gone. Core, Appearance, Shortcuts, Skills, Workspace, Local model, Advisor, Privacy and Diagnostics are all one click away in a flat list.
- Skills tab is one simple list: the bucket grid and its coaching text are replaced by a single filterable list of shared skills with full names, codex/pinned/excluded tags and Pin/Exclude actions. The review inbox and install hooks stay tucked away in their folds.
- No more white-screen crashes: a crashed view or dialog (for example Impact fed partial data by an older sidecar) now shows a contained "try again" fallback instead of taking the whole app down.
- Leftovers removed: the half-removed Sources, Sessions and Memories panels are fully gone, along with the two dead Core toggles that controlled them and the advisor example cards.

## Install

1. Download the `Spine_0.5.5_universal.dmg` asset from GitHub Releases.
2. Open the DMG and drag Spine to Applications.
3. Launch Spine and restart your CLI if it was already running.

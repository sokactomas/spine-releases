# Spine v0.5.2

Released: 2026-06-22

## What's new

This is a performance and battery-life release — no new features, just a noticeably smoother, quieter app.

- Smoother knowledge graph: the graph now animates through direct drawing instead of re-rendering the whole scene on every frame, so opening and settling a large graph no longer stutters or lags.
- Quieter when idle: Spine's background work now backs off when nothing is changing. The CLI-surface sync, permission learning, and team sync all slow their cadence on an idle machine instead of running on a fixed timer all day — fewer fan spin-ups and less battery drain when you step away.
- Less wasted work: the guardian only scans your notes when an active pane has actually produced output recently, and diagnostics polling pauses while the window is hidden.
- Snappier UI: the sidebar stops re-rendering on unrelated updates, panel-resize no longer stutters, and very large documents open without blocking.

## Install

1. Download the `Spine_0.5.2_universal.dmg` asset from GitHub Releases.
2. Open the DMG and drag Spine to Applications.
3. Launch Spine and restart your CLI if it was already running.
